# Telehealth Doctor Profile

This project is a simple Telehealth Doctor Profile webpage created using HTML and CSS.  
It displays doctor details, emergency alerts, and available appointment slots with a clean and professional medical-style design.

---

 Features

- Doctor profile section
- Emergency alert highlighting
- Appointment slots displayed in a table
- Clean blue and white medical theme
- Separate CSS file for styling

---

 Technologies Used

- HTML
- CSS
- VS Code
- GitHub
